# 🛒 E-Commerce Sales Analytics Dashboard

## 📌 Project Overview

This project analyzes an E-Commerce Sales dataset to uncover valuable business insights, identify sales trends, understand customer purchasing behavior, and predict future sales using Machine Learning. An interactive Streamlit dashboard was developed to visualize KPIs, sales trends, and predictions.

---

## 🎯 Objectives

- Import and explore the dataset
- Clean missing values and duplicate records
- Perform Exploratory Data Analysis (EDA)
- Analyze sales trends and customer behavior
- Build Machine Learning models
- Compare model performance
- Predict future sales
- Develop an interactive Streamlit dashboard

---

# 🤖 Machine Learning

The following regression models were trained and compared:

- Linear Regression
- Decision Tree Regressor ⭐ (Best Model)
- Random Forest Regressor

### Evaluation Metrics

- MAE (Mean Absolute Error)
- MSE (Mean Squared Error)
- RMSE (Root Mean Squared Error)
- R² Score

The **Decision Tree Regressor** achieved the best performance and was selected as the final model for predicting sales.

# 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Ecommerce-Sales-Analytics.git
```

Go to the project folder:

```bash
cd Ecommerce-Sales-Analytics
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the Streamlit application:

```bash
streamlit run app.py
```
